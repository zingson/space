# 微服务

服务介绍。


## 注册中心

表名：ms_discover
字段：服务名、IP、port、过期时间、更新时间、
      name   ip  port  expires  updatedAt
      
客户端心跳：定时心跳更新过期时间，超时由TTL索引剔除服务      
客户端监听：更新事件、删除事件 
     
## 配置中心

表名：ms_config
字段：服务名、配置JSON内容、备注、更新时间、创建时间

客户端监听：配置更新


## ws消息服务

表名： ms_ws

## ms-smm

运营系统用户管理、角色配置、功能菜单、权限管理


## ms-cms

包含 栏目、文章 管理
