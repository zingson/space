# 淘宝客

IDEA 开发环境，需要指定编译输出目录与工作目录在同一个目录，这样才能使用相对路径访问资源文件。

Nginx接收请求URL：`http://xxx.mz7z.com/index.html`
网关接收请求URL： `http://127.0.0.1:5888/tbk/index.html?xxx=tmallshop`
服务接收请求URL： `http://127.0.0.1:80/index.html?xxx=tmallshop`
